What is TweakCommunity?

TweakCommunity (TC) is a search engine for packages used in jailbreaking Apple devices.

All the tweaks used in TC can be used in Sileo Cydia Zebra via TC,
all using the original download links, and all information is automatically updated so that viruses cannot be inserted by management intervention (we can't afford to do that on our servers)

TC is not OSS, except for the front end, but it does expose an API that can be accessed by bots, applications, etc. See https://api.amania.jp/docs for more information
If you find a cracked tweak or a tweak with legal issues on TC, please contact us.
If you want to register a Repository on TC, please contact us.
After confirming the repository, we will add the repository and after a while, the bot will access your repository.
